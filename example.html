JSLists - A simple way to make HTML lists collapsible

This is an example of using JSLists to create a collapsible list with multiple levels of nesting.

Becasue JSLists doesn't effect any element of the HTML list you are passing is, you can use all of the functionality of standard HTML list.

So ALL list items can contain any valid HTML elements, such as anchors <a>, <img>, <div> etc.

In this example, I have included the links to the amazing FontAwesome CSS library

At present, you will have to manually add these.

Calling JSLists is simple. In the example below, we simply pass the ID of the list we want to collapse:
<script>JSLists.applyToList('simple_list', 'ALL');</script>
